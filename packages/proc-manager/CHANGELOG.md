# @toolbox/proc-manager

## 1.1.0

### Minor Changes

- [`8ed978c`](https://github.com/tux86/toolbox/commit/8ed978cc2af634ade58f0db4ec294b25b8b363c6) Thanks [@tux86](https://github.com/tux86)! - Add proc-manager - Interactive TUI for managing system processes and ports

  Features:

  - Combined view showing processes with CPU%, memory, and listening ports
  - Ports view for listening TCP ports
  - Detail view with Kill (SIGTERM) and Force Kill (SIGKILL) options
  - Auto-refresh toggle (2s interval)
  - Filter by name, PID, port, or user
